## A estrutura

Para encontrar o código que você deseja basta navegar dentro dos capítulos que estão divididos em pastas

```

    |___ exemplo1.php
    |___ exemplo2.php
capitulo-2
    |___ exemplo1.php
    |___ exemplo2.php
```
